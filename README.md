# 設計模式 Design Pattern

# 安裝 Installation

1. 透過 `composer` 來安裝相關依賴。
```shell
$ composer install
```

2. 輸入指令 `phpunit` 來執行測試。
```shell
$ ./vendor/bin/phpunit tests
```
